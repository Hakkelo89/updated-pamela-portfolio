Quiz

In this project, I created a timed quiz using Javascript fundamentals which stores high scores, so that progress can be compared to my peers and viewed on a highscores page. The quiz contains 5 questions. Once the user starts the quiz, the timer is automatically initiated and the first question with 5 answer choices appear on the screen. If the user clicks on a correct, they move onto the next question. If they click on a incorrect answer, they will not move onto the next question and 5 second will be deducted from the remaining time for completing the quiz. Once they have completed the quiz, the user can enter their initials onto the scoreboard. Skills used: HTML layout, CSS styling, Javascript
